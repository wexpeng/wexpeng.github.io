夜游湘江

发表于 2018-08-09 | 分类于 阅历 |
| 字数统计: 974 字
|  湘江边

…

真不知该如何动笔……

今日，是好朋友的生日。她说饭店消费了4000多一点，酒吧消费了快1000。尽管在吃饭的时候喝了两瓶好酒，还是执意去酒吧蹦迪。

我不可避免想起前几天另一个朋友生日，我们暑假在学校研究东西，生日其实并不在意。他下午买了两个好吃的西瓜，晚上买了一个很大的蛋糕，三个宿舍的人在实验室11点半才回宿舍，蛋糕吃的很开心：祝愿项目顺利。那是真的很开心。

今日，灯红酒绿，醉意迷离。同样的11点半，还没有回去的打算，陆陆续续嗨到两点，大家各自回家。我还想自己租共享汽车回去，突然想起自己喝了酒。成年的现实感扑面而来，过些日子，我是否就是这样的生活。又想到宿管阿姨应该不会给我开门，路边有种比白天还要热闹的氛围，扮丑或卖萌直播的，有锤男朋友女朋友胸口的，更多是蹲在街边抽烟的，还有一两个唱歌的小姐姐，唱的真难听，不过长得挺好看的。

这种割裂感在我眼前蔓延。

漫无目的的在路边游荡，同这里的每个行人一样，唯一不同的是，我是独身一人。

孤独有诸多的好处，比如现在，没有人打扰我去看他们和她们。

我走到了人迹罕至的一些地方，这里只有长椅，和睡着的人、和他们在这个城市的战利品——一些瓶瓶罐罐。我也想寻找一个长椅，走了这么久有些累了，不如与他们一起领略这个城市的残忍，我走啊走，没有找到剩下的长椅了。我突然很想哭，这群无家可归的人，冬天住在哪里？

只隔着一条街，那一边醉生梦死，这一边宵衣旰食。

辉黄的路灯照耀着些许人，其实我并非第一次遇见，我也不明白今夜，我的心为何颤抖。父亲曾经和我说，刚刚南下打工，那是冬天，没有工作没有钱，又没有脸面找包子店要个馒头，路过菜市场，捡到一坨白豆腐，陪他过了一个年。他说的云淡风轻，我也笑笑不说话。我从来不敢同情无家可归之人，因为我父亲，算得上妻离子散，而我，孤身一人四五年，过得也很开心。

杜甫曾言：南村群童欺我老无力，因为茅屋被秋风所破。推己及人。安得广厦千万间，大庇天下寒士俱欢颜！

我突然想做一名记者，在各个城市记录辛苦的人民。他们同我们一样，都有血有肉。不同的是，他们无家可归。他们在想什么？想离自己千里之遥的儿子？想已经离异或者改嫁的妻子？会不会抱怨城市残酷，会不会伤心睡长椅的如今？












照片这么模糊因为我怕被他们发现。

回到路边，那个唱歌的小姐姐依旧在唱，唱一些得不到的被偏爱的。我路过肯德基，两个上夜班的小伙子在趴桌子上睡觉，这里很空调很凉快，突然好奇，这里面这么舒适，那些以艰难生计而睡长椅的人，为何没有一个在这里，倒是有几个衣着光鲜的女生趴在桌子上，大概是隔壁酒吧喝醉又不能回家。来此醒酒了。

我要了一份皮蛋瘦肉粥和一杯清水。看了时间，4点半，窗外，马路上零星已有一些环卫工人。
